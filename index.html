Endpoints
GET /health
Service liveness probe. Returns model status, version, and feature names.
POST /predict
Single prediction from JSON body. Returns diagnosis label, confidence score, and per-class probabilities.
POST /predict Content-Type: application/json { "Pregnancies": 5, "Glucose": 166, "BloodPressure": 72, "SkinThickness": 19, "Insulin": 175, "BMI": 25.8, "DiabetesPedigreeFunction": 0.587, "Age": 51 } → 200 OK { "status": "success", "prediction": "Diabetic", "prediction_code": 1, "confidence": 0.87, "probabilities": { "not_diabetic": 0.13, "diabetic": 0.87 } }
POST /predict-batch
Multipart CSV upload (field name: file). Returns JSON results + summary. Add ?download=true to get a CSV response.
GET /model-info
Returns model type, kernel, support vector counts, scaler statistics, and test accuracy.
GET /sample-input
Returns a ready-to-use sample payload for /predict.
